# 🎬 Cinema Booking System (Python CLI Project)

A simple command-line cinema booking system built with Python.  
This project simulates a basic movie ticket reservation system with user authentication, seat selection, booking management, and a time-based booking restriction system.

---

## 🚀 Features

- 👤 User Registration & Login system  
- 🎟️ Movie listing with available showtimes and dates  
- 💺 Seat selection (5x5 cinema hall simulation)  
- ❌ Booking cancellation system  
- ⛔ 15-day booking restriction after making a reservation  
- 🔔 Background notification service using threading  
- 📝 File-based logging system (audit trail of actions)

---

## 🛠️ Technologies Used

- Python 3
- threading module
- datetime module
- file handling (txt logging)
- os module

---
